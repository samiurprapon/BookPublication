### Book Publication
  
  ## Abstract
  >*Book Publication is java swing based GUI application to keep records.*
  
  
  ## Installation
  Use IntelliJ Idea or  
  
  Manually  go to ```/src``` and open ```terminal/cmd```   
  Compile: 
  
  ```bash
  javac *.java
  ```
  
  run.
  ```bash 
  java Main
  ```
